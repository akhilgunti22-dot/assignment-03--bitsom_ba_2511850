# Student Performance Analysis & Prediction

## Overview
This project analyses student data and predicts whether a student will pass or fail using a machine learning model.

## Dataset
The dataset contains 15 students with subject scores, attendance, study hours, and pass/fail status.

## Tasks
- Data analysis using pandas  
- Visualizations using Matplotlib and Seaborn  
- Logistic Regression model for prediction  
- Feature importance analysis  

## Outputs
- 5 Matplotlib plots  
- 2 Seaborn plots  
- 1 Feature importance plot  

## How to Run
Open the notebook and run all cells to see outputs and plots.

## Note
The dataset is small, so accuracy may vary. The focus is on understanding the workflow.
