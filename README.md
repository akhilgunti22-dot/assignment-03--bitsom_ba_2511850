
# Restaurant Menu & Order Management System

This project is a Python-based Restaurant Order Management System built using core data structures such as lists, dictionaries, and nested data.

## Features

### Task 1: Menu Exploration
- Display menu grouped by categories
- Count total and available items
- Identify most expensive item
- List items under ₹150

### Task 2: Cart Operations
- Add items to cart with quantity handling
- Prevent duplicate entries (update quantity instead)
- Remove items from cart
- Generate order summary with GST calculation

### Task 3: Inventory Management
- Deep copy inventory using copy.deepcopy()
- Simulate order fulfilment
- Handle insufficient stock
- Generate reorder alerts

### Task 4: Sales Log Analysis
- Calculate daily revenue
- Identify best-selling day
- Find most ordered item
- Add new sales data dynamically
- Print all orders using enumerate

##  Technologies Used
- Python (Core concepts only)
- Lists
- Dictionaries
- Loops & Conditionals
